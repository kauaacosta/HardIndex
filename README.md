# HardIndex - Catálogo de Hardware de Alta Performance

Este projeto foi desenvolvido como parte do meu portfólio de Ciência da Computação (Estácio). O objetivo é oferecer uma interface limpa e interativa para entusiastas de hardware.

## Tecnologias e Ferramentas
* Frontend:** HTML5 e CSS3 (Design Responsivo).
* Cloud & Hosting:** AWS Amplify.
* Segurança:** Implementação de boas práticas de diretórios para deploy seguro.

## Deploy na Nuvem
O site está hospedado oficialmente na AWS, utilizando uma arquitetura Serverless para garantir que o carregamento seja rápido de qualquer lugar.

## O que aprendi
Neste projeto, pude praticar o ciclo de vida de uma aplicação: do código local até o deploy em um ambiente de nuvem real, lidando com configurações de faturamento, segurança e entrega de arquivos.
